# Install

- use the way you like to install

## Linux (recommend)

```sh
curl -fsSL https://raw.githubusercontent.com/luoluoter/jetson-examples/main/install.sh | sh
```

## Github (for Developer)

```sh
git clone https://github.com/luoluoter/jetson-examples
cd jetson-examples
pip install .
```
